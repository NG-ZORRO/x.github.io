API#

nx-prompts#

PropertyDescriptionTypeDefault
[nxItems]包含多个提示项的列表NxPrompt[]-
[nxTitle]显示在提示列表顶部的标题string-
[nxVertical]设置为 true 时, 提示列表将垂直排列booleanfalse
[nxWrap]设置为 true 时, 提示列表将自动换行booleanfalse
(nxItemClick)提示项被点击时的回调函数EventEmitter<{ data: PromptProps }>-

interfaces#

NxPrompt#

PropertyDescriptionTypeDefault
key唯一标识用于区分每个提示项string-
disabled设置为 true 时禁用点击事件booleanfalse
icon提示图标显示在提示项的左侧string-
label提示标签显示提示的主要内容string-
description提示描述提供额外的信息string-
children嵌套的子提示项NxPrompts[]-
